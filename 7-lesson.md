## Завдання

### 1. Фрукти

Створи масив `fruits` з елементами: `"Apple"`, `"Pear"`, `"Orange"`.

- Виведи в консоль останній елемент.
- Заміни значення `"Pear"` на `"Lemon"`.
- Додай у кінець масиву `"Grapes"`.

### 2. Черга

Створи масив `styles` з елементами: `"Jazz"`, `"Blues"`.

1. Додай у кінець `"Rock-n-Roll"`.
2. Заміни значення посередині на `"Classics"`.
3. Видали перший елемент масиву та виведи його.
4. Додай на початок `"Rap"` та `"Reggae"`.

### 3. Сума чисел

Напиши функцію `sumInput()`, яка просить користувача вводити числа (через `prompt`) і зберігає їх у масив.

- Закінчує запитувати, коли користувач вводить не число, натискає Cancel або нічого не вводить.
- Після цього виводить суму цих чисел.

### 4. Пошук у масиві

Є масив об'єктів (користувачів):

```javascript
let users = [
  { id: 1, name: "Ivan" },
  { id: 2, name: "Petro" },
  { id: 3, name: "Maria" },
];
```

Знайди користувача з `id: 2`, використовуючи метод `find`.

### 5. Фільтрація та трансформація

Є масив чисел: `[1, -2, 16, 8, -5, 25]`.

1. Використовуй `filter`, щоб залишити тільки додатні числа.
2. Використовуй `map`, щоб добути квадратний корінь (`Math.sqrt`) з кожного числа отриманого списку.
