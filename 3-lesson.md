## Задачі на `&&`, `||`, `!`

### 1. Чи має доступ користувач?

```javascript
let isAdmin = true;
let isLoggedIn = false;
```

Виведи "Доступ дозволено", якщо користувач або адмін, або просто увійшов.

### 2. Вік для купівлі алкоголю

```javascript
let age = 17;
```

Виведи "Можна купити алкоголь", якщо користувачу 18 і більше, інакше — "Заборонено".

### 3. Логін і пароль

```javascript
let login = "admin";
let password = "12345";
```

Виведи "Вхід дозволено", якщо логін — "admin" і пароль — "12345". Інакше — "Доступ заборонено".

### 4. Число в межах

```javascript
let number = 15;
```

Перевір, чи входить число в діапазон від 10 до 20 включно.
Виведи "У межах" або "Поза межами".

### 5. Не порожнє ім’я

```javascript
let name = "";
```

Виведи "Привіт, [ім'я]", якщо `name` не порожнє. Інакше — "Ім’я не вказано".

### 6. Перевірка на парність та додатність

```javascript
let number = 6;
```

Виведи "Число парне і додатне", якщо це справді так.

### 7. Чи сьогодні вихідний?

```javascript
let day = "Saturday";
```

Виведи "Вихідний", якщо день — субота або неділя.

### 8. Не авторизований

```javascript
let isLoggedIn = false;
```

Виведи "Будь ласка, увійдіть", лише якщо користувач не авторизований (використай `!`).
