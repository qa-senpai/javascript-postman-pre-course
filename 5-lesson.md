## Завдання

### 1. Привітання

Напиши функцію `sayHello(name)`, яка приймає ім'я і виводить у консоль: `"Привіт, [name]!"`.
Виклич її для кількох різних імен.

### 2. Сума двох чисел

Створи функцію `sum(a, b)`, яка повертає (`return`) суму двох чисел.
Виведи результат роботи функції в консоль: `console.log(sum(5, 10));`.

### 3. Перевірка віку

Напиши функцію `checkAge(age)`.

- Якщо вік понад 18, поверни `true`.
- Інакше поверни `false` (або `confirm('Батьки дозволили?')` як у прикладі).

### 4. Мінімум з двох чисел

Напиши функцію `min(a, b)`, яка повертає менше з двох чисел `a` та `b`.

```javascript
min(2, 5) == 2;
min(3, -1) == -1;
min(1, 1) == 1;
```

### 5. Стрілочні функції

Перепиши функцію з завдання 2 (`sum`), використовуючи синтаксис стрілочної функції (`=>`).
